Ocean Beats was the first project I joined during the FIU ShellHacks. We used C+ and Unity to create an educational music game. This game mixes my passion as a musician and educator, and I learned to plan and develop a project and build in a short time frame. This was a special project that helped me to see the hard work necessary to learn and to develop a project from the ground up.
