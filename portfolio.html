Self Reflection on Portfolio Project

With this project I was able to dig in alot more with building out a figma project and creating a more detailed design rather than using it only for individual elements. It was interesting to take a page with built in components in Figma and transfer them over to HTML and CSS. My goal was to emulate a similar scenario to what a web developer would typically be presented with by having the full design laid out before beginning to work on the HTML. Another main focus of mine was to really to focus more on the accessability aspect of the app. It is important to me that anyone could use it and find value.

When going through the testing I found one interesting scenario that I had not yet considered. I deployed my portfolio on github in a seperate repo with out the readMe file or any other affiliation with this course. When I opened the application on my wife's phone. The pages were all translated to french. Which significantly impacted the design since some of the words were much larger than the english translation and did not fit within the original container. So in the future I would like to test out specifically spanish,french and the other top Languages that are common to the majority of users.
